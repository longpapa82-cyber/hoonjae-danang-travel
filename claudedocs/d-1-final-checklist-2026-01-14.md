## 🌐 2. 오프라인 데이터 사전 캐싱

**절차**:
1. 앱 접속 (WiFi 연결 상태에서)
   ```
   https://hoonjae-danang-travel.vercel.app
   ```

2. 날씨 탭 클릭하여 5일 예보 확인
   - 2026-01-15 (목) ~ 2026-01-19 (월) 날씨 로드

3. 각 날짜별로 상세 날씨 정보 확인

4. 개발자 도구로 캐시 확인 (선택사항)
   - Chrome DevTools → Application → Local Storage
   - `weather_cache_*` 키 확인
   - `weatherCache_*` 키 확인 (인메모리 캐시는 확인 불가)

### 2.2 지도 데이터 확인

**목적**: Google Maps 데이터 사전 로드 및 오프라인 지도 다운로드

2. 각 일차별로 경로 확인

3. Google Maps 앱에서 오프라인 지도 다운로드

### 2.3 PWA 리소스 캐싱

## 📱 3. PWA 설치 및 권한 확인

### 3.1 홈 화면에 추가 (PWA 설치)

**Android**:
1. Chrome 브라우저에서 앱 접속

4. 홈 화면 아이콘으로 앱 실행 테스트

**iOS**:
1. Safari 브라우저에서 앱 접속
2. 하단 공유 버튼 → "홈 화면에 추가"
3. "다낭 여행 트래커" 아이콘 확인
4. 홈 화면 아이콘으로 앱 실행 테스트

**검증**:
- [ ] 홈 화면에 PWA 아이콘 추가됨
- [ ] 아이콘 클릭 시 전체 화면 모드로 실행됨 (주소창 없음)
- [ ] 앱 아이콘과 스플래시 스크린 정상 표시

### 3.2 위치 권한 확인

**절차**:
1. 앱 실행 후 "위치 권한 허용" 모달 확인
2. "위치 권한 허용" 버튼 클릭
3. 브라우저 권한 요청 팝업에서 "허용" 선택
4. 현재 위치가 지도에 파란색 마커로 표시되는지 확인

**권한 재확인 (Android)**:
- 설정 → 앱 → Chrome → 권한 → 위치 → "앱 사용 중에만 허용"

**권한 재확인 (iOS)**:
- 설정 → Safari → 위치 → "앱 사용 중"

**검증**:

- [ ] 현재 위치가 지도에 표시됨

## 🚨 문제 해결 가이드

### 백업 파일 생성 실패

**증상**: "백업 파일 생성 및 다운로드" 클릭 시 아무 일도 일어나지 않음

**해결**:
1. 브라우저 콘솔 확인 (개발자 도구 → Console)
2. localStorage가 비어있는지 확인
3. 다른 브라우저에서 시도 (Chrome, Safari)
4. 시크릿 모드가 아닌지 확인 (시크릿 모드는 localStorage 제한됨)

### 오프라인 모드에서 앱이 로드되지 않음

**증상**: WiFi 끄고 접속 시 "인터넷 연결 없음" 오류

**해결**:
1. Service Worker 재등록 필요
2. 온라인 상태에서 앱 접속 → 새로고침 여러 번
3. 개발자 도구 → Application → Service Workers → "Update on reload" 체크
4. 새로고침 후 오프라인 테스트 재시도

### GPS 위치가 부정확함

**증상**: 현재 위치가 실제 위치와 다름

**해결**:
1. GPS 정확도 높이기: 설정 → 위치 → 정확도 높음
2. WiFi 및 블루투스 켜기 (위치 정확도 향상)
3. 건물 밖으로 나가서 GPS 신호 재수신
4. 위치 권한이 "앱 사용 중에만 허용"으로 설정되었는지 확인

### 날씨 정보가 표시되지 않음

**증상**: 날씨 탭이 비어있거나 로딩 중

**해결**:
1. 온라인 상태 확인 (첫 로드는 인터넷 필요)
2. API 키 확인 (.env.local 파일)
3. 브라우저 콘솔에서 에러 메시지 확인
4. 캐시 만료 시 재로드 (24시간 캐시)
