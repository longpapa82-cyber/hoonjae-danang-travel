## 📱 Android 기기 테스트 (Chrome/Samsung Internet)

### 1단계: PWA 앱 설치

**방법 1: 자동 설치 프롬프트**
1. Chrome 브라우저로 https://hoonjae-danang-travel-090723s-projects.vercel.app 접속
2. 화면 하단에 나타나는 "앱으로 설치" 배너 클릭
3. "설치" 버튼 확인

**방법 2: 수동 설치**
1. Chrome 브라우저 메뉴 (⋮) 열기
2. "홈 화면에 추가" 선택
3. 이름 확인: "다낭여행"
4. "추가" → "자동으로 추가" 선택

**확인 사항**:
- [ ] 홈 화면에 "다낭여행" 아이콘 생성됨
- [ ] 아이콘 디자인이 올바름 (비행기/야자수 이미지)
- [ ] 아이콘을 길게 눌러 앱 정보 확인 가능

---

### 2단계: Standalone 모드 실행

1. 홈 화면에서 "다낭여행" 아이콘 터치
2. 앱이 전체화면으로 실행되는지 확인

**확인 사항**:
- [ ] 브라우저 주소창이 보이지 않음 (Standalone 모드)
- [ ] 상태바만 표시됨 (시간, 배터리, 네트워크)
- [ ] 하단 브라우저 내비게이션 바가 없음
- [ ] 앱이 네이티브 앱처럼 보임

- [ ] 날씨 캐시 시간 표시 ("5분 전" 등)

**3-2. 지도 탭**

- [ ] 지도 줌인/줌아웃 동작
- [ ] 지도 드래그 동작

**3-3. 일정 탭**
- [ ] 5일 일정 표시
- [ ] 일별 타임라인 표시
- [ ] 활동 카드 클릭 시 상세 정보
- [ ] 이미지 클릭 시 모달 확대 보기

**3-4. 설정 탭**

- [ ] 테마 전환 애니메이션 부드러움

**3-5. 하단 네비게이션**
- [ ] 4개 탭 모두 터치 가능
- [ ] 탭 전환 애니메이션 부드러움
- [ ] 현재 탭 하이라이트 표시
- [ ] 터치 피드백 있음

### 4단계: 오프라인 모드 테스트 ⚠️ 중요!

**4-1. 사전 캐싱**
1. Wi-Fi 연결 상태에서 앱 실행
2. 모든 탭 한 번씩 방문 (홈/지도/일정/설정)
3. 날씨 정보 로딩 확인
4. 지도 로딩 확인

**4-2. 오프라인 전환**
1. 휴대폰 설정 → 비행기 모드 ON
2. 또는 Wi-Fi/모바일 데이터 모두 OFF
3. 네트워크 연결이 완전히 끊겼는지 확인

**4-3. 오프라인 테스트**
- [ ] 앱이 계속 실행됨 (크래시 없음)
- [ ] 화면 상단에 "오프라인" 알림 표시
- [ ] 홈 탭: 여행 정보 표시됨
- [ ] 일정 탭: 일정 표시됨
- [ ] 날씨: 마지막 캐시된 정보 표시 ("캐시된 정보" 표시)
- [ ] 설정: 다크모드 토글 동작
- [ ] 지도: 로딩 실패 메시지 또는 마지막 캐시 표시

**4-4. 온라인 복구**
1. 비행기 모드 OFF 또는 Wi-Fi 재연결
2. 네트워크 복구 대기 (5-10초)

**확인 사항**:
- [ ] "온라인" 알림 표시 (녹색)
- [ ] 날씨 정보 자동 업데이트
- [ ] 지도 정상 로딩

### 5단계: GPS 위치 추적 테스트

**5-1. 위치 권한 허용**
1. 앱 최초 실행 또는 "위치 추적 시작" 버튼 클릭
2. 위치 권한 요청 팝업 확인
3. "앱 사용 중에만 허용" 또는 "이번만 허용" 선택

**5-2. 위치 추적 동작**
- [ ] 현재 위치 카드에 좌표 표시
- [ ] 위치 정확도 표시 (± Xm)
- [ ] 실외에서 GPS 정확도 확인 (10-50m)
- [ ] 실내에서 GPS 정확도 확인 (50-500m, 부정확할 수 있음)

**5-3. 지도 탭에서 확인**
- [ ] 지도에 현재 위치 표시 (파란색 점)
- [ ] 현재 위치로 지도 자동 이동
- [ ] GPS 정확도 원(circle) 표시

**5-4. 배터리 소모 확인**
- [ ] 위치 추적 중 배터리 소모 정상 범위

## 🔧 일반적인 문제 해결

### PWA 설치가 안 돼요

**Android**:
- Chrome 버전 확인 (최신 버전 권장)
- 시크릿 모드가 아닌지 확인
- 저장 공간 확인 (최소 100MB)
- 브라우저 캐시 삭제 후 재시도

**iOS**:
- Safari 사용 확인 (Chrome 불가)
- iOS 버전 확인 (16.4 이상 권장)
- 저장 공간 확인
- Safari 설정 → 웹사이트 데이터 삭제

---

### 오프라인 모드가 작동 안 해요

1. 온라인 상태에서 모든 탭 방문했는지 확인
2. Service Worker 등록 확인:
   - Android: Chrome → 메뉴 → 도구 → 개발자 도구 → Application → Service Workers
   - iOS: 확인 불가, 앱 재설치 시도
3. 캐시 저장 공간 확인 (최소 50MB)

---

### GPS 위치가 부정확해요

1. 위치 서비스 활성화 확인
2. 고정밀도 모드 설정 (GPS + Wi-Fi + 모바일)
3. 실외에서 테스트 (건물 내부는 부정확)
4. 몇 분 대기 (GPS 위성 연결 시간 필요)

---

### 앱이 느려요

1. 백그라운드 앱 종료
2. 기기 재시작
3. 앱 재설치
4. 캐시 삭제 후 재설치
