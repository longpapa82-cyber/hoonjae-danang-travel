홈 화면

메인 대시보드 - 여행 진행 상황 한눈에 확인

주요 컴포넌트

  • CurrentLocationCard - 현재 위치 표시
  • RouteInfoCard - 다음 목적지까지 경로 정보
  • MapView - Google Maps 통합
  • TravelProgress - 진행률 원형 차트

상태별 표시

  • 여행 전: D-day 카운트다운
  • 여행 중: 현재 활동 및 진행률
  • 여행 후: 완료 메시지

핵심 기능

실시간 진행 상황

현재 시간을 기준으로 어떤 일정을 진행 중인지 자동 계산하여 표시

위치 기반 정보

사용자의 현재 위치와 다음 목적지까지의 거리, 예상 소요 시간 제공

시각적 진행률

원형 프로그레스 바로 전체 여행 일정 대비 현재 진행 상황을 직관적으로 표시
