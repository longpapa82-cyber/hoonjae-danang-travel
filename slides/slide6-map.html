지도 화면

실시간 위치 추적 및 경로 안내

지도 기능

  • Google Maps 통합 - 실시간 지도 표시
  • 위치 추적 - Geolocation API 활용
  • 마커 표시 - 여행지 위치 마킹
  • 경로 그리기 - Polyline으로 이동 경로
  • 편의시설 - 주변 호텔/편의점/마트/카페

편의시설

  • 호텔 시설, 24시간 편의점
  • 대형마트, 카페

사용자 인터랙션

위치 추적 토글

버튼 클릭으로 실시간 위치 추적 활성화/비활성화

편의시설 필터

카테고리별 편의시설 표시/숨기기. 바텀시트로 상세 목록 확인 가능

마커 클릭

지도의 마커를 클릭하면 InfoWindow로 장소 정보 표시
